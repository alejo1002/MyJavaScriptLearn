Data Types

  • JavaScript Types are dinamic: The same variable can be user to hold different data types.
  • Primitive Data: The typeof operator can return primitive types:.
    • string
    • number
    • boolean
    • null
    • undefined: a variable with no value
  • Complex Data: The typeof operator can return complex types:
    • function
    • object: even arrays
  • Null: Something that doesn's exist.
    • In JavaScript, the data type of null is an object.
    • var person = null; value: null, type: object
    • var person = undefined; value and type are undefined
