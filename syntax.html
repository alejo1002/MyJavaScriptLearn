JavaScript syntax

  • JavaScript is a programing languaje
  • JavaScript programs are executed by the web browser.
  • Statements are composed of: Values, Operators, Expressions, Keywords and Comments .
    • Values:
      • Literals: Numbers and Strings
      • Variables: store data values (var keyword)
    • Operators:
      • Arithmetic operators ( + - * / )
      • Assignment operator ( = )
    • Expressions: is a combination of values, variables, and operators, which computes to a value.
    • Keywrods: are used to identify actions to be performed.
    • Comments: Comments are ignored, and will not be executed (// or between /* and */)
    • Identifiers:are used to name variables(and keywords, and functions, and labels).
    • Case Sensitive: JavaScript does not interpret VAR or Var as the keyword var
    • Camel Case: Lower Camel Case
